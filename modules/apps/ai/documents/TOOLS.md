# Tools

## Available Tools

The following tools are available on this system:

### Shell
Full access to the host shell (fish). Can run any command the user can.

### Git
Manage repositories, create commits, push changes. The user's primary flake config is at ~/.flake.

### Nix
NixOS system management. Use `just` commands for common operations (build, switch, test, fmt).

### Summarize
Summarize web pages, PDFs, and YouTube videos.
